# 医院管理系统 python947

本项目是基于 Django 框架开发的医院管理系统，配备了完整的技术文档。系统主要包括患者、医生和管理员三种角色，旨在实现医院工作流程的高效管理。

## 技术栈

- 后端：Python 3.x，Django
- 数据库：SQLite（默认），兼容其他数据库
- 前端：HTML，CSS，JavaScript，Bootstrap

## 功能模块

### 管理员

- **科室管理**：展示科室列表，包括科室基本信息。
- **医生管理**：查看医生列表，添加新医生，并分配科室。
- **药品管理**：维护药品列表，添加新药品。
- **患者管理**：管理现有患者信息，协助新患者注册。

### 医生

- **接诊房间**：查看待诊患者列表，限定医生只能接诊所属科室的患者。
- **就诊处理**：完成就诊，录入医嘱，开立药品。
- **就诊记录**：回顾历史就诊记录和药品清单。

## 系统角色

- 患者：进行注册，查看个人就诊记录。
- 医生：处理接诊事宜，管理患者就诊记录。
- 管理员：维护系统基础信息，包括科室、医生和药品数据。

## 运行环境

- Python 环境要求：3.6 或以上版本
- Django 环境要求：2.x 或兼容版本
- 数据库：SQLite（默认），可选支持 MySQL、PostgreSQL 等

## 部署步骤

1. 环境准备：安装 Python 和 Django。
2. 数据库配置：根据需求选择数据库，并进行配置。
3. 代码部署：将代码部署到服务器。
4. 运行：通过命令行启动 Django 服务。
